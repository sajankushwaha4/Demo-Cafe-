# ☕ Kushwaha Cafe Management System

A modern **Cafe Management System** developed using **Java, Spring Boot, HTML, CSS, JavaScript, SQL, and Python**. This project helps manage cafe operations such as menu management, customer orders, reservations, user authentication, and administration through a responsive web application.

## 📋 Prerequisites

Before running the project, make sure the following are installed on your system:

* JDK 17 or later
* Visual Studio Code (or IntelliJ IDEA)
* Extension Pack for Java (VS Code)
* Maven (optional, Maven Wrapper is included)
* Git

### Recommended VS Code Extensions

* Extension Pack for Java
* Language Support for Java by Red Hat
* Debugger for Java
* Maven for Java
* Project Manager for Java

## ▶️ How to Run the Project

### Method 1: Using VS Code

1. Clone the repository.

```bash
git clone https://github.com/your-username/your-repository.git
```

2. Open the project in **Visual Studio Code**.

3. Make sure the **Java Extension Pack** is installed.

4. Open:

```text
src/main/java/com/kushwahacafe/cafe/CafeApplication.java
```

5. Click the **Run** button above the `main()` method, or press **F5**.

---

### Method 2: Using Maven

Open the terminal inside the project folder and run:

**Windows**

```bash
mvnw spring-boot:run
```

or

```bash
.\mvnw spring-boot:run
```

**Linux / macOS**

```bash
./mvnw spring-boot:run
```
